[Learning Notes] Quo Vadis, Action Recognition? A New Model and the Kinetics Dataset

Posted on 2017-09-27 Edited on 2017-10-16 In Learning Notes

背景：
在足够大的数据集上训练动作分类网络，当应用于不同的时间任务或数据集时，性能将是否会得到类似的提升，这是视频领域的open question。 构建视频数据集的挑战意味着，大多数用于动作识别的人口基准很小，大约有10k个视频。在本文中，我们的目标是使用新的Kinetics Dataset视频数据集[16]来提供这个问题的答案，该数据集比以前的数据集HMDB-51 [18]和UCF-101 [27]大两个数量级。

我们的实验策略是从文献重新实现一些代表性的神经网络架构，然后首先对Kinetics Dataset进行预训练，然后对HMDB-51和UCF-101进行fine-tuning。 结果表明，pre-training总是能提高performance，但是随着体系结构的类型，提升空间也会有很大的不同。 基于这些发现，我们引入了一种新模型，能够利用Kinetics进行pre-training后的优势，实现较好的performance。 该模型被称为“Two-Stream Inflated 3D ConvNets”（I3D），建立在最先进的图像分类架构之上，但将其过滤器和汇集内核（以及可选的其参数）充满3D，导致非常深入， 自然时空分类器。 基于Inception-v1 [13]的I3D模型在对Kinetic进行预训练后，获得的性能远远超过了最先进的技术。

2. Two-Stream

Two-Stream Convolutional Networks for Action Recognition in Videos提出的是一个双流的CNN网络，分别捕捉空间和时间信息。空间和时间网络分别给出动作的分类结果，然后把结果融合，使用取平均值或者SVM的方法（实验中显示SVM准确率更高），得到最终结果

3. 3D-ConvNet

3D Convolutional Neural Networks for Human Action Recognition
传统的2D卷积方法是用一个2维的卷积层对特征图进行采样，从而得到下一层的特征图。那么3D就是额外增加了一个时间维度，即对一定数量帧的图片，用同一个3维的卷积层去采样，从而得到下一维的特征图。
