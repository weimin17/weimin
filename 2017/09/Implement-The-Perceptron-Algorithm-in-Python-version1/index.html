Secondly, when updating weights and bias, comparing two learn algorithms: perceptron rule and delta rule.** (Actually Delta Rule does not belong to Perceptron; I just compare the two algorithms.) It turns out that the algorithm performance using delta rule is far better than using perceptron rule. +** Perceptron Rule ** Perceptron Rule updates weights only when a data point is misclassified. Weightvector is incremented by (LearningRate)(xj):
Δw = (learning rate) (inputvector) weights[i] += l_rate * row[i]
+** Delta Rule ** Delta rule updates weights when the output would be correct. uses Error-->(Actual - Predicted) before the predicted value goes through the threshold in the learning rule to decide weight update:
Δw = (learning rate) (Actual - Predicted)(inputvector)
