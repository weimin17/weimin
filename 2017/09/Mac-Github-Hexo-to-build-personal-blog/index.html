Mac: Using Github+Hexo to build personal blog(1)

Posted on 2017-09-09 Edited on 2020-01-11 In Computer Skills

本文介绍了在Mac环境下如何利用GitHub和Hexo搭建个人博客，网上有不少教程已经过时，或者有些坑。本人花了点时间搭建好第一个个人主页，决定将自己调试流程总结如下。主要包括：Mac下安装、配置Github+Hexo的方法，以及如何利用其构建个人主页。

![upload successful](/images/pasted-0.png)

一、利用GitHub创建仓库

创建网站不仅需要域名，而且需要网站空间。网站空间也叫虚拟主机，是用来存放网站内容的地方。但是租借虚拟主机需要很大的开销，幸运的是，我们可以把静态网站托管到GitHub上，GitHub是一个面向开源及私有软件项目的托管平台。

1.创建代码库

首先需要登陆到GitHub，如果没有GitHub账号，可以使用邮箱注册，然后点击GitHub页面右上角的加号，选择New repository创建新仓库，仓库名为：用户名.github.io，这里命名格式为固定的，用户名是你的GitHub账号用户名，比如我的仓库名为”weimin17.github.io”：

正确创建后，可以看到： ![upload successful](/images/pasted-3.png) __注：红框部分必须跟你的github用户名完全相同。__ 建好之后，我们就拥有了用于搭建个人博客的网站空间。 ![upload successful](/images/pasted-1.png) ### 2.GitHub Pages 是什么 GitHub Pages 可以被认为是用户编写的、托管在 GitHub 上的静态网页。 ### 3.GitHub Pages 的优势 • 免费 • 无限流量 • 全世界理想的访问速度 ### 4.GitHub Pages 注意点 1.
