Brief Introductions of Selected Research | Weimin Lyu (Lv)

Posted on 2018-12-04 Edited on 2018-12-05

My research interests are Machine Learning, Deep Learning, Natural Language Processing, and Computer Vision. Application areas include Topic Models (e.g., topic evolutionary analysis based on Dynamic Topic Model and Latent Dirichlet Allocation), Combination of Computer Vision and NLP (e.g., pathology image classification assisting with NLP classifier, Video Question & Answering), Object Detection (e.g., Industry-level helmet detection), Telecommunications Prediction (e.g., telecommunication trend nowcasting using convolutional LSTM, LSTM, SVR, ARIMA algorithms), Recommending System (e.g., recommending scientific research collaborators using link prediction and machine learning), etc. The following information introduces briefly selected research of mine:

1 Combination of Computer Vision and Natural Language Processing

1.1 Pathology Image Classification Assisting with NLP Classifier

The Final Goal is to classify pathology images, with the help of text descriptions to improve classification performance. The project is still in progress, and I have already finished the following work:

  1. Use PdfMiner and PyPdf2 to extract text and images from pdf files Here

  2. NLP analysis: Implement text mining tutorials – preprocessing and word2vec in Gensim and NLTK Here

  3. Img2Vec: Use pre-trained models (or do none pre-trained) in PyTorch to extract vector embeddings from any image and calculate their similarity Here

The whole project is available in https://github.com/weimin17/Image-NLP-classification .

1.2 Video Question & Answering

This project is a quick implement using “convolutional sequence to sequence learning” framework on video question and answering, trying to combining descriptions(text) and video features in order to get a better video QA system. However, the research did not proceed because of my advisor’s interest change.

2 Natural Language Processing

2.1 Topic Models

This research project focuses on topic evolutionary analysis based on Dynamic Topic Model(DTM) and Latent Dirichlet Allocation(LDA), which is a 3 layer Bayesian Model. I was engaged as the person in charge of the project and carried out independent study, which is important for a PhD student. The task is not only to reveal the scientific structure based on LDA, but also to excavate the mechanism of scientific evolution based on the time series. I designed a scientific evolutionary analysis method system using iDTM, which is an innovative attempt in the Information Science area. I applied the system into NSF foundation project data as an empirical study. I also tried to compare it with Deep Belief Nets, implementing by Keras framework. During the research, I found I am eager to obtain a deeper understanding of computer science and acknowledge more advanced computing technology.

3 Computer Vision

3.1 Object Detection

This repository is one of my projects during my three-month-internship in K2Data. I am the main contributor in a 3-people team, and the achievement has been applied in over 2,000 factories. It tries to verify whether workers wear safety helmets when working via CCTV camera videos. The real-time video requires both high accuracy and faster calculating speed.

  • Research, design, and implement applications for object detection

  • Increased performance of helmet detection to 90% in industry level

  • Main contributor in a 3-people team, achievement has been applied in over 2,000 factories

The codes are shared on My Github

4 Machine Learning & Deep Learning

I have built up basic Deep Learning skills by learning and implementing algorithms, such as CNN, RNN, LSTM, Encoder-Decoder, Seq2Seq, one-shot learning, reinforcement learning. Some learning notes are posted on my blog.

4.1 Telecommunications Prediction

This project aims to predict high-dimensional telecommunications by proposing a novel model based on Convolutional LSTM, considering both temporal and spatial factors. I implemented and compared LSTM, ConvLSTM, SVR and ARIMA model to predict future telecom trends, coding with TENSORFLOW framework and other Python packages. Codes and results are uploaded to My Github

4.2 Recommending System

One of my master’s project achievements was published in the top 5 journals in the Information Science area in China. It’s an application on data analysis combining Machine Learning and Link Prediction, in order to predict the potential scientific collaborators. I use Extremely Randomized Trees to extract and rank features. After experimenting over 20 different link prediction algorithms, the prediction accuracy of the top 10 results reached 100% at national and city level. I also found a novel way to promote efficiency of the program running time from over 24 hours to less than 3 hours.
