1.What is Machine Learning

Artificial Intelligence 是目标，Machine Learning是手段，Deep Learning 是Machine Learning的一种方法。

1.1 “Fake” Artificial Intelligence

生物的行为取决于两件事：1）先天本能 2）后天学习。Machine Learning的先天本能相当于是人类提前设计好的规则，即hand-craft rules，机器根据这个提前设计好的规则去做事情，这并非真正的AI。比如说有很多chat-bot，输入“turn off the music”与“do not turn off the music”，机器只会根据提前设置好的关键字turn off来做出判断，结果是上述两种输入都导致相同的行为：turn off the music。

hand-craft rules的缺点是，很难考虑到所有的可能性-这也就导致他没有办法超越他的创造者人类。他只能根据已有的规则做事情，遇到人类想不到的规则，机器就不知道该怎么办。
另一个例子，来自Yann Lecun的FB， 意思是如果拆解开所谓的智能机器人，内部是一堆IF命令，也就是提前设计好的hand-craft rules。

1.2 Real Artificial Intelligence

真正的人工智能是让Machine 有后天学习的能力，而不仅仅是靠先天本能（hand-craft rules）。比如给他足够的图片资料学习，他学习后能识别出，新给一张图片，图片中是cat/dog/monket, etc。

Technically speaking, Machine Learning是在寻找一个function，让机器具有一种能力：根据提供的资料（数据），机器寻找出我们需要的function。比如

Speech recognition:

$$\f（音频信号） = “How are you.”$$

Image recognition:
f（图片） = “cat”

2. Machine

Framework （以supervised learning为例）

1]先准备一个包含成千上万function的function set（即models）

2]再根据training data，会有function input，function output，与真实的output比较，能知道function是好还是不好。

3]function set中的functions数量可能无穷尽，我们需要一个好的策略从众多function中挑选出最好的function $f^$, 根据这个最好的$f^$来预测结果：$f^*$（新图片）= 预测结果。

总结而言：

step1：模型假设，选择模型框架（linear regression）

step2：模型评估，如何判断众多模型的好坏（loss function）

step3：模型优化，如何筛选最优的模型（gradient descent）

3. Scenario


Supervised Learning：大量training data与ground truth labels

Semi-supervised Learning：少量有labeled data与大量unlabeled data

Transfer Learning：只有少量labeled data，但有大量不相关的data（can be both labeled and unlabeled），这些不相关的data能给我们的任务带来什么帮助。

Unsupervised Learning：只有unlabeled data，如根据已有图片生成新图片。

Reinforcement Learning：

Supervised Learning-知道正确答案，learn from teacher

Reinforcement Learning-没有正确答案，只知道最终的结果是好还是不好，但不知道哪里不好，然后从这些反馈的评价、结论中学习。learn from critics。

AlphaGo：结合了supervised learning（先从棋谱中学习）与reinforcement learning（两台机器互相下棋，从输赢结果继续学习）

关于不同的scenario理解，这里总结的很详细。
